lsof -i tcp:4006

ps aux \| grep \(compile command\)

### Mac文件夹路径权限

1. 首先 获得核心权限，输入密码  sudo -s
2. 进入到该文件夹目录 cd /Applications/文件夹地址 列出该文件夹文件所有权限 ls -l
3. 更改权限，比如777 chmod 777 文件夹目录
4. 搞定。ls -l 查看文件权限是否更改成功。
