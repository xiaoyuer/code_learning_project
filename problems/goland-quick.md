# Goland quick



| **快捷键** | **作用** |
| :--- | :--- |
| ctrl + n | 导航到类名 |
| ·ctrl + shift +n | 编辑器全局查找文件或者文件夹 |
| ctrl + e | 打开最近的文件夹或者文件 |
| ctrl + shift + i | 查看源码 |
| ctrl + shift + m | 重构表达式为函数，光标在函数末尾 |
| ctrl + shift + a | 格式化代码，规范代码 |
| ctrl + i | 自动写出接口所有方法 |
| ctrl + p | 在函数括号里面使用，参数就显示 |
| alt + j | 相同单词多选，是一个个的，不是全部 |
| ctrl + shift + / | 多行注释 |
| ctrl + b | 打开光标处的结构体定义位置 |
| ctrl + - | 对代码进行折叠 |
| ctrl + r | 替换文本 |
| ctrl + f | 查找文本 |
| ctrl + shift + f | 进行全局查找 |
| ctrl + g | 快速定位到输入的行 |
| shift +enter | 即使光标在中间，下方插入新的行 |
| ctrl + x | 删除当前光标所在的行 |
| ctrl + shift + u | 大小写转化 |
| alt + 7 | 打开文件方法列表 |
